## Getting Started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the app:

   ```bash
   npx expo start
   ```

3. **Connect your music server** in Settings:
   - Enter your server URL, username, and password
   - Test the connection to verify it works
   - Start browsing and streaming your music!
